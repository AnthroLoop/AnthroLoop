# AnthroLoop

AI自律型組織によるデジタルビジネス。

## 概要

ソロプレナー（CPO: 人間）× AI C-suite で年間売上1億円を目指す実験的プロジェクト。

## AI組織構造

| 役職 | 起動 | 責任 |
|------|------|------|
| CEO | `/ceo` | 戦略、OKR |
| CTO | `/cto` | 技術、アーキテクチャ |
| CFO | `/cfo` | 財務、価格戦略 |
| CMO | `/cmo` | マーケティング、グロース |
| COO | `/coo` | オペレーション、自動化 |
| Board | `/board` | 部門横断会議 |
| Sprint | `/sprint` | スプリント管理 |

## コマンド

| コマンド | 用途 |
|---------|------|
| `/standup` | 朝会レポート |
| `/weekly-review` | 週次レビュー |
| `/okr-check` | OKR進捗確認 |

## クイックスタート

```bash
cd ~/AnthroLoop
claude  # Claude Code を起動
# /board でプロダクト決定会議を開催
```

# /okr-check - OKR進捗確認

現在のOKRの進捗を全役職の視点から確認し、レポートしてください。

## 手順

1. `docs/strategy/okr/` から最新のOKRファイルを読む
2. 各KRの進捗を評価（数値ベース）
3. 各役職の貢献状況を確認
4. リスクと対策を特定

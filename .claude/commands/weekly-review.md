# /weekly-review - 週次レビュー

今週の全部門の活動を振り返り、次週の計画を策定してください。

## 手順

1. 各部門の `docs/` を読み、今週の進捗を把握
2. `.claude/rules/revenue-targets.md` のKPIと照合
3. OKR進捗を確認
4. 次週のアクションを部門ごとに策定
